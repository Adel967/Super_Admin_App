تطبيق الإدارة الرئيسية (Super Admin App)

وثيقة المواصفات الفنية والوظيفية التفصيلية

1️⃣ شاشات المصادقة والأمان

1.1 شاشة تسجيل الدخول

  • حقول: البريد الإلكتروني، كلمة المرور
  • رابط "نسيت كلمة المرور"
  • زر تسجيل الدخول
  • التحقق من صلاحيات المستخدم

1.2 شاشة استعادة كلمة المرور

1.3 شاشة الملف الشخصي

  • عرض معلومات المسؤول
  • تعديل البيانات الشخصية
  • تغيير كلمة المرور
  • إعدادات الأمان
  • سجل تسجيل الدخول

2️⃣ لوحة التحكم الرئيسية (Dashboard)

2.1 الشاشة الرئيسية

إحصائيات عامة:

  • عدد الشركات المسجلة (الإجمالي/النشطة/المعلقة/الملغاة)
  • عدد الحجاج والمعتمرين الكلي
  • عدد البرامج النشطة
  • إجمالي الإيرادات

رسوم بيانية:

  • مخطط نمو عدد الشركات شهرياً
  • مخطط توزيع الحجاج والمعتمرين
  • مخطط الإيرادات الشهرية

عناصر إضافية:

  • إشعارات النظام
  • تنبيهات هامة (اشتراكات منتهية، طلبات معلقة)

3️⃣ إدارة الشركات

3.1 شاشة قائمة الشركات

جدول يعرض:

  • اسم الشركة
  • رقم الترخيص
  • تاريخ التسجيل
  • حالة الحساب (نشط/معلق/ملغى)
  • نوع الاشتراك
  • تاريخ انتهاء الاشتراك
  • عدد البرامج
  • عدد الحجاج/المعتمرين
  • رصيد الحساب

فلاتر البحث:

  • حسب الاسم
  • حسب الحالة
  • حسب نوع الاشتراك
  • حسب تاريخ التسجيل
  • حسب المدينة

أدوات إضافية:

  • خيارات الترتيب
  • تصدير البيانات (Excel, PDF, CSV)

3.2 شاشة إضافة شركة جديدة

معلومات الشركة الأساسية:

  • اسم الشركة (عربي/إنجليزي)
  • رقم السجل التجاري
  • رقم الترخيص السياحي
  • تاريخ الترخيص
  • تاريخ انتهاء الترخيص
  • نوع الشركة (حج/عمرة/كلاهما)

معلومات الاتصال:

  • العنوان الكامل
  • المدينة/المحافظة
  • الدولة
  • الرمز البريدي
  • رقم الهاتف الأرضي
  • رقم الجوال
  • البريد الإلكتروني
  • الموقع الإلكتروني

المعلومات المالية:

  • رقم الحساب البنكي
  • اسم البنك
  • IBAN
  • رقم السجل الضريبي

رفع المستندات:

  • السجل التجاري (PDF/صورة)
  • الترخيص السياحي
  • شهادة التسجيل الضريبي
  • صورة الهوية للمسؤول
  • شعار الشركة

معلومات المسؤول:

  • اسم المسؤول الكامل
  • رقم الهوية
  • رقم الجوال
  • البريد الإلكتروني
  • الصلاحيات

إعدادات الحساب:

  • اسم المستخدم
  • كلمة المرور المؤقتة
  • نوع الاشتراك
  • تاريخ بدء الاشتراك
  • مدة الاشتراك
  • الحد الأقصى للبرامج
  • الحد الأقصى للحجاج
  • زر الحفظ والإرسال بالبريد الإلكتروني

3.3 شاشة تفاصيل الشركة

تبويبات متعددة:

  • التبويب الأول: المعلومات العامة
    • عرض جميع معلومات الشركة
    • زر التعديل
    • سجل التعديلات
    • حالة الحساب مع إمكانية التغيير
  • التبويب الثاني: الاشتراك
    • تفاصيل الاشتراك الحالي
    • تاريخ البدء والانتهاء
    • المميزات المتاحة
    • سجل الاشتراكات السابقة
    • أزرار: تجديد الاشتراك، ترقية الاشتراك، إيقاف الاشتراك، تعليق الاشتراك مؤقتاً
  • التبويب الثالث: البرامج
    • قائمة جميع برامج الشركة
    • حالة كل برنامج
    • عدد المسجلين في كل برنامج
    • إحصائيات البرامج
  • التبويب الرابع: الحجاج والمعتمرين
    • العدد الإجمالي
    • قائمة بالحجاج/المعتمرين
    • تفاصيل كل حاج/معتمر
    • حالة التسجيل
    • حالة السفر
  • التبويب الخامس: المالية
    • إجمالي المدفوعات
    • المدفوعات المعلقة
    • سجل الفواتير
    • تقارير مالية تفصيلية
    • رسوم بيانية للإيرادات
  • التبويب السادس: الإحصائيات
    • معدل استخدام النظام
    • عدد المستخدمين النشطين
    • معدل الحجوزات
    • تقييمات العملاء
    • رسوم بيانية تحليلية
  • التبويب السابع: سجل النشاطات
    • جميع الأنشطة والعمليات
    • تاريخ ووقت كل عملية
    • المستخدم المنفذ
    • نوع العملية

3.4 شاشة تعديل بيانات الشركة

  • نفس حقول شاشة الإضافة
  • عرض البيانات الحالية
  • إمكانية التعديل الجزئي
  • سجل التغييرات
  • تأكيد التعديل

3.5 شاشة إدارة الاشتراكات

إنشاء باقات الاشتراك:

  • اسم الباقة
  • السعر
  • المدة
  • عدد البرامج المسموحة
  • عدد الحجاج المسموح
  • المميزات المتاحة
  • قيود الاستخدام

قائمة باقات الاشتراك:

  • عرض جميع الباقات
  • تعديل الباقات
  • تفعيل/إلغاء الباقات
  • عدد الشركات المشتركة في كل باقة

3.6 شاشة تجديد/إيقاف الاشتراكات

  • قائمة الاشتراكات القريبة من الانتهاء
  • تنبيهات الاشتراكات المنتهية
  • إجراءات سريعة:
    • تجديد تلقائي
    • تجديد يدوي
    • إيقاف مؤقت
    • إلغاء نهائي
  • سجل العمليات

4️⃣ التقارير والإحصائيات

4.1 شاشة التقارير الشاملة

فلاتر التقارير:

  • نوع التقرير
  • الفترة الزمنية (من-إلى)
  • الشركة المحددة أو الكل
  • نوع البرنامج (حج/عمرة)
  • حالة البيانات

أنواع التقارير المتاحة:

  • تقرير الشركات:
    • عدد الشركات المسجلة
    • توزيع الشركات حسب المدن
    • الشركات النشطة/غير النشطة
    • معدل التجديد
    • رسم بياني للنمو
  • تقرير الحجاج والمعتمرين:
    • العدد الإجمالي
    • التوزيع حسب الدول
    • التوزيع حسب الجنسيات
    • التوزيع حسب الفئات العمرية
    • التوزيع حسب الجنس
    • رسوم بيانية تفصيلية
  • تقرير البرامج:
    • عدد البرامج الكلي
    • البرامج النشطة/المكتملة
    • معدل الإشغال
    • أكثر البرامج طلباً
    • مقارنة الأسعار
  • التقرير المالي:
    • إجمالي الإيرادات
    • الإيرادات حسب الشركة
    • الإيرادات حسب الفترة
    • المدفوعات المعلقة
    • المبالغ المستردة
    • رسوم بيانية مالية
  • تقرير الأداء:
    • معدل استخدام النظام
    • أكثر المستخدمين نشاطاً
    • معدل رضا العملاء
    • متوسط التقييمات

خيارات التصدير:

  • PDF
  • Excel
  • CSV
  • طباعة مباشرة
  • إرسال بالبريد الإلكتروني

5️⃣ سجل النشاطات

5.1 شاشة سجل النشاطات

  • سجل كامل لجميع العمليات في النظام
  • فلترة حسب:
    • نوع العملية
    • المستخدم
    • التاريخ والوقت
    • القسم
  • تفاصيل كل عملية
  • إمكانية التصدير

6️⃣ إدارة الإشعارات

6.1 شاشة إنشاء إشعار

أنواع الإشعارات:

  • إشعار عام لجميع المستخدمين
  • إشعار للشركات فقط
  • إشعار للحجاج/المعتمرين فقط
  • إشعار للموظفين
  • إشعار مخصص لمجموعة محددة

محتوى الإشعار:

  • العنوان (عربي/إنجليزي)
  • النص (عربي/إنجليزي)
  • نوع الإشعار (معلومة/تحذير/عاجل)
  • إرفاق صورة/ملف
  • رابط URL (اختياري)

خيارات الإرسال:

  • إرسال فوري
  • جدولة الإرسال (تاريخ ووقت محدد)
  • إرسال متكرر (يومي/أسبوعي/شهري)

قنوات الإرسال:

  • إشعار داخل التطبيق
  • البريد الإلكتروني
  • رسالة SMS
  • الكل

6.2 شاشة الإشعارات الذكية

إنشاء إشعارات مستهدفة حسب:

  • اهتمامات المستخدمين
  • سلوك المستخدمين
  • موقع المستخدمين
  • نوع البرنامج المسجل فيه
  • حالة التسجيل
  • تاريخ السفر
  • الخدمات المستخدمة

القوالب الجاهزة:

  • إشعار تذكير بموعد السفر
  • إشعار تجديد الاشتراك
  • إشعار انتهاء البرنامج
  • إشعار توفر خدمة جديدة
  • إشعار تقييم الخدمة

6.3 شاشة سجل الإشعارات

  • قائمة جميع الإشعارات المرسلة
  • حالة كل إشعار:
    • عدد المستلمين
    • عدد من فتح الإشعار
    • عدد من تفاعل مع الإشعار
    • معدل النجاح
  • إحصائيات الإشعارات
  • إمكانية إعادة الإرسال

7️⃣ الإعدادات

7.1 شاشة الإعدادات العامة

إعدادات النظام:

  • اسم المنصة
  • الشعار
  • معلومات الاتصال
  • العملة الافتراضية
  • المنطقة الزمنية
  • تنسيق التاريخ

إعدادات الأمان:

7.2 شاشة إدارة الصلاحيات

  • إنشاء أدوار جديدة
  • تحديد صلاحيات كل دور:
    • قراءة
    • كتابة
    • تعديل
    • حذف
    • الوصول للتقارير
  • ربط المستخدمين بالأدوار

8️⃣ إدارة المسؤولين

8.1 شاشة قائمة المسؤولين

جدول المسؤولين:

  • الاسم
  • البريد الإلكتروني
  • الدور
  • تاريخ الإضافة
  • آخر تسجيل دخول
  • الحالة (نشط/معلق)
  • أزرار الإجراءات

8.2 شاشة إضافة مسؤول جديد

  • الاسم الكامل
  • البريد الإلكتروني
  • رقم الجوال
  • اسم المستخدم
  • كلمة المرور
  • الدور والصلاحيات
  • القسم
  • زر الإضافة
